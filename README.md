# AI Product Recommendation System

This project is a simple AI-powered product recommendation system built using **React + Vite**.

The application allows users to enter natural language preferences (e.g. *"laptop under $1000"*) and displays recommended products from a dynamic product list.

---

##  Features
- Dynamic product list fetched from DummyJSON API
- Natural language preference handling
- Category detection (laptops, smartphones, skincare, groceries, etc.)
- Budget-based filtering (under / above price)
- Clean and responsive UI
- Modular and maintainable code structure

---

##  Tech Stack
- React
- Vite
- JavaScript (ES6)
- DummyJSON API
